# Full Character Creation Core v1.0

Two simplified nodes:

- Character Blueprint Creator v1.0
- Character Shot Planner v1.0

The Blueprint Creator separates face, upper body, lower body, bust, markings, and default clothing. Bust size uses visual descriptors from Very Small through Overly Large. Heritage is a dedicated field separate from skin tone.

The Shot Planner sends only crop-relevant traits to Krea or Qwen. Face-only shots do not receive bust, waist, buttocks, or lower-body anatomy.
